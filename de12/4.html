＜使い方＞

難易度が３段階に分かれており、難易度順に問題を漢字の読みを当てていくというものになります。

  1. difficulty_levels = {
  2.     1:{"反片":"はんぺん","蒲鉾":"かまぼこ"},
  3.     2: {"御手洗団子": "みたらしだんご", "玉蜀黍": "とうもろこし"},
  4.     3: {"鹿尾菜": "ひじき", "陸蓮根": "おくら"},
  5.     }
  7. def quiz(difficulty):
  9.     words = difficulty_levels.get(difficulty, {})
  11.     if not words:
  12.         print("その難易度は存在しません。")
  13.         return
  15.     
  16.     for word, reading in words.items():
  17.         user_input = input(f"{word} の読み方は？ ")
  19.         if user_input == reading:
  20.             print("正解です！")
  21.         else:
  22.             print(f"残念、正解は {reading} でした。")
  24. for difficulty in range(1, 4):
  25.     print(f"\n難易度 {difficulty}:")
  26.     quiz(difficulty)
